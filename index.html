Tiago is an assistant professor at UMass Boston. Tiago's research helps programmers write software with fewer bugs.

Tiago develops tools that localize errors, proves the correctness of algorithms, and mines how we write code to identify anomalies.

Tiago is looking for self-motivated students who are interested in improving the quality of our software.

Projects

Safer GPU programming

Localizing bugs caused by data-races & over-synchronization

Safer parallel runtimes

Proving the correctness of parallel runtimes and testing implementations

Large-scale bug localization

Data-mining source code to identify statistically-anomalous code
